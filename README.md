## 📡 Mutual Coupling Reduction in MIMO Patch Antenna Array for Wi-Fi and Bluetooth Applications

### Overview

This project focuses on designing a compact **1×2 MIMO Microstrip Patch Antenna Array** operating at **2.4 GHz** for Wi-Fi and Bluetooth applications. The primary challenge addressed is **mutual coupling** between closely spaced antenna elements, which negatively affects antenna isolation, gain, and overall communication performance.

To overcome this issue, various decoupling techniques were studied and implemented. A parasitic decoupling structure was introduced between antenna elements to improve isolation while maintaining compact antenna dimensions suitable for modern wireless devices.

### Key Features

* 📶 Designed a 2.4 GHz Microstrip Patch Antenna for ISM band applications.
* 📡 Developed a compact 1×2 MIMO antenna array.
* 🔬 Performed extensive literature review on mutual coupling reduction techniques.
* ⚡ Implemented parasitic decoupling structures to improve antenna isolation.
* 📈 Analyzed S-parameters, gain, and radiation characteristics using HFSS.
* 🛠️ Fabricated and experimentally validated the final antenna design.
